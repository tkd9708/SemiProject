# 맨도롱소랑
2021.01.09 ~ 2021.02.15

## 프로젝트 소개
한 곳의 사이트에서 제주도의 관광명소와 주변 맛집을 소개하고, 회원 추천식당, 관광지 리뷰 등을 통하여 다양한 정보를 공유하고 자신만의 일정을 기록하여 여행객의 수월한 관광을 돕는 웹 사이트

## Team
* **PL** 김상희<br/>
* **Front-end** 김소연, 양호준, 정수진<br/>
* **Back-end** 김상희, 박근령, 오상근<br/>

## Stack
* **기술** : AWS, MySQL, JavaScript, Java, AJAX, JSP, jQuery, json, xml, HTML5, CSS 
* **Open API** : JEJU Tour API, Kakao map api
* **Tool** : Eclipse, MySQL workbench, SourceTree, Tomcat, Quantum, FileZila
